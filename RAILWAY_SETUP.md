# Інструкція: Налаштування PostgreSQL на Railway

## Проблема
Зараз при кожному рестарті додатку дані зникають, бо використовується SQLite (файлова база), яка не зберігається між деплоями на Railway.

## Рішення: Додати PostgreSQL

### Крок 1: Додати PostgreSQL плагін
1. Відкрий свій проект на Railway (https://railway.app)
2. Натисни на свій проект
3. Натисни кнопку "+ New" або "Add Service"
4. Вибери "Database" → "Add PostgreSQL"
5. Railway автоматично створить базу даних

### Крок 2: Перевірка змінної DATABASE_URL
1. Відкрий свій основний сервіс (не PostgreSQL, а твій додаток)
2. Перейди в розділ "Variables"
3. Має бути змінна `DATABASE_URL` з значенням типу:
   `postgresql://postgres:password@hostname:5432/railway`

Якщо її немає:
1. Натисни "New Variable"
2. Reference Variable
3. Вибери PostgreSQL → DATABASE_URL

### Крок 3: Перезапуск
1. Після додавання PostgreSQL натисни "Deploy" або зачекай автоматичного деплою
2. При першому запуску створяться всі таблиці і адмін користувач
3. Тепер дані будуть зберігатися навіть після рестартів!

### Перевірка
Після налаштування:
- Створи тестову групу
- Перезапусти додаток (Settings → Restart)
- Група має залишитися

## Логін після налаштування
- Логін: `admin`
- Пароль: `admin123`

Після першого входу рекомендую змінити пароль!
