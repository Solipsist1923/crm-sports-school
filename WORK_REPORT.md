Повністю робочий REST API на FastAPI з усіма необхідними функціями.

**Структура:**
```
backend/
├── app/
│   ├── api/
│   │   ├── auth.py          # Аутентифікація
│   │   ├── students.py      # Управління учнями
│   │   ├── attendance.py    # Відвідування
│   │   ├── payments.py      # Оплати
│   │   └── stats.py         # Статистика
│   ├── core/
│   │   ├── config.py        # Конфігурація
│   │   ├── database.py      # Підключення до БД
│   │   └── security.py      # JWT, хешування
│   ├── models/
│   │   └── models.py        # SQLAlchemy моделі
│   └── schemas/
│       └── schemas.py       # Pydantic схеми
├── main.py                  # FastAPI додаток
├── init_db.py              # Ініціалізація БД
├── requirements.txt        # Залежності
├── start.bat              # Скрипт запуску
└── README.md              # Документація
```

**Реалізовані API endpoints:**

**Аутентифікація:**
- POST `/api/auth/login` - Вхід в систему
- GET `/api/auth/me` - Інформація про користувача

**Учні:**
- GET `/api/students` - Список з фільтрами
- GET `/api/students/{id}` - Деталі учня
- POST `/api/students` - Створити
- PUT `/api/students/{id}` - Оновити
- DELETE `/api/students/{id}` - Видалити

**Відвідування:**
- GET `/api/attendance` - Список
- GET `/api/attendance/date/{date}` - За датою
- GET `/api/attendance/student/{id}` - Історія учня
- POST `/api/attendance` - Відмітити
- PUT `/api/attendance/{id}` - Оновити
- DELETE `/api/attendance/{id}` - Видалити

**Оплати:**
- GET `/api/payments` - Список
- GET `/api/payments/overdue` - Прострочені
- GET `/api/payments/student/{id}` - Історія учня
- POST `/api/payments` - Створити
- PUT `/api/payments/{id}` - Оновити
- DELETE `/api/payments/{id}` - Видалити

**Статистика:**
- GET `/api/stats/dashboard` - Дашборд
- GET `/api/stats/attendance` - Відвідуваність

**Особливості:**
- JWT аутентифікація
- Ролі: admin (повний доступ), trainer (тільки свої учні)
- Автоматичне списування занять з абонемента
- Валідація всіх даних
- Автодокументація (Swagger)
- CORS для frontend

## Як запустити проєкт

### Backend:
```bash
cd backend
pip install -r requirements.txt
python init_db.py
uvicorn main:app --reload
```
Або просто `start.bat` на Windows.

Backend: http://localhost:8000
API Docs: http://localhost:8000/docs

### Frontend:
Відкрити `frontend/pages/login.html` у браузері.

### Тестові дані:
- Admin: `admin` / `admin123`
- Trainer: `trainer1` / `trainer123`
