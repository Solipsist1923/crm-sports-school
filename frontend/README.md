# CRM для спортивної школи - Frontend

## Структура

```
frontend/
├── pages/
│   ├── login.html       # Сторінка входу
│   ├── dashboard.html   # Головна сторінка (дашборд)
│   ├── students.html    # Список учнів (TODO)
│   ├── attendance.html  # Відвідування (TODO)
│   ├── payments.html    # Оплати (TODO)
│   └── groups.html      # Групи (TODO)
├── css/
│   └── style.css        # Стилі
├── js/
│   ├── config.js        # Конфігурація API
│   ├── api.js           # API функції
│   ├── auth.js          # Логіка авторизації
│   └── dashboard.js     # Логіка дашборду
└── components/          # Компоненти (TODO)
```

## Запуск

1. Переконайтеся, що backend запущений на `http://localhost:8000`

2. Відкрийте `pages/login.html` у браузері або використайте локальний сервер:

```bash
# Python
python -m http.server 8080

# Node.js
npx http-server -p 8080
```

3. Відкрийте http://localhost:8080/pages/login.html
